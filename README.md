# AtliQ-Hardware-Sales-Analysis-Excel-Project-
Project Overview

This project analyzes the sales and financial performance of AtliQ Hardware using Microsoft Excel. The objective of the project is to transform raw business data into meaningful insights that help understand sales growth, customer contribution, product performance, and market trends.

Using Excel tools such as Pivot Tables, Data Cleaning, and Visualization, multiple reports and dashboards were created to evaluate business performance between 2019 and 2021.

This project simulates a real-world business analysis scenario, helping stakeholders make data-driven decisions.

1️⃣ Customer Performance Analysis

This report evaluates customer-wise sales growth from 2019 to 2021.

Key Insights:

Major retailers like Amazon, Flipkart, and AtliQ Exclusive contribute a significant portion of total revenue.

Many customers experienced over 200% sales growth between 2020 and 2021.

Helps identify high-value customers for strategic partnerships.
